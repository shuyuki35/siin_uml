「死印 青き終焉」の情報を整理するため＆応援の意味も込めて、試しに作ってみた。  
絵心も音楽的才能もないけれど、コーディングなら多少はできるじゃないかと、ふと閃いた結果の産物。  

「ここはこうではないか」という指摘、大歓迎。
